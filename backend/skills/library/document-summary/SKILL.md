# Document Summary

分析和总结文档，提取关键信息、主要论点和重要细节。适用于处理长文档、报告或文章。

## Workflow

1. **快速浏览**: 初步了解文档结构
   - 识别文档类型和格式
   - 浏览标题、目录、小节
   - 估计文档长度和复杂度

2. **深度阅读**: 仔细理解核心内容
   - 识别主题和中心论点
   - 标记关键信息和数据
   - 理解论证逻辑和结构

3. **信息提取**: 提取关键要素
   - 核心观点和结论
   - 重要数据和事实
   - 关键论据和证据

4. **结构组织**: 组织摘要结构
   - 确定摘要层次
   - 安排信息优先级
   - 建立逻辑连贯性

5. **摘要生成**: 撰写结构化摘要
   - 开头概述核心要点
   - 中间详述关键内容
   - 结尾总结意义影响

6. **质量检查**: 验证摘要质量
   - 确保信息准确
   - 检查遗漏要点
   - 优化表达清晰度

## Summary Types

| 类型 | 长度 | 适用场景 |
|------|------|----------|
| 一句话摘要 | 1-2句 | 快速了解主旨 |
| 执行摘要 | 3-5句 | 商务决策参考 |
| 详细摘要 | 1-2段 | 深入理解内容 |
| 结构化摘要 | 分点列举 | 系统性梳理 |

## Guidelines

- 保持客观中立，忠于原文
- 区分事实和作者观点
- 使用清晰简洁的语言
- 保留关键数据和引用
- 标注不确定或有争议的内容

## Output Format

### 标准摘要模板

```markdown
## 文档摘要

**文档**: [标题]
**类型**: [报告/论文/文章等]
**长度**: [页数/字数]

### 核心要点
- 要点1
- 要点2
- 要点3

### 主要内容
[详细摘要内容]

### 关键数据
- 数据1
- 数据2

### 结论/建议
[作者的结论或建议]
```

## Examples

```
输入: 一份50页的市场研究报告

输出:
## 文档摘要

**文档**: 2025年AI Agent市场研究报告
**类型**: 市场研究报告
**发布**: 2025年1月

### 核心要点
- AI Agent市场规模预计2025年达到XXX亿美元
- 主要增长驱动力：企业自动化需求、技术成熟度提升
- 竞争格局：头部厂商占据60%市场份额

### 主要发现
1. 市场增长：同比增长45%，超出预期
2. 技术趋势：多Agent协作成为主流架构
3. 应用场景：客服、代码开发、数据分析为三大场景

### 关键数据
- 市场规模：XXX亿美元（2025年预测）
- 年复合增长率：35%
- 企业采用率：从15%增长到40%

### 报告建议
建议企业在2025年加大AI Agent投入，重点关注...
```

## Safety Checks

- 不添加原文没有的信息
- 不改变原文的立场和观点
- 对于敏感内容标注警示
- 保护文档中的隐私信息

## Success Criteria

- 摘要准确反映原文核心内容
- 信息完整，无重要遗漏
- 结构清晰，便于快速理解
- 长度适中，详略得当
- 语言简洁，表达准确
