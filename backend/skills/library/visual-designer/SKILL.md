# Visual Designer

创作视觉设计、布局和图形元素。适用于UI/UX设计、图形创作、视觉构图规划或品牌视觉开发。

## Core Competencies

- **视觉设计**: 创造美观且功能性的视觉方案
- **用户体验**: 设计直观易用的交互体验
- **品牌视觉**: 建立一致的视觉识别系统
- **信息架构**: 组织和呈现复杂信息
- **响应式设计**: 适配多种设备和场景

## Workflow

1. **需求理解**: 深入理解设计目标
   - 明确设计目的和受众
   - 了解品牌调性和限制
   - 收集参考和灵感

2. **信息架构**: 规划内容结构
   - 梳理内容层次
   - 设计导航流程
   - 确定信息优先级

3. **概念设计**: 探索设计方向
   - 创建情绪板（Mood Board）
   - 探索多种视觉方向
   - 确定设计风格

4. **布局设计**: 规划视觉结构
   - 设计栅格系统
   - 规划页面布局
   - 确定视觉层次

5. **细节设计**: 完善设计元素
   - 选择配色方案
   - 设计UI组件
   - 添加图标和图形

6. **交付迭代**: 完成设计交付
   - 制作设计规范
   - 准备切图资源
   - 根据反馈迭代

## Design Principles

### 视觉层次

| 层级 | 元素 | 处理方式 |
|------|------|----------|
| 主要 | 标题、核心CTA | 最大、最醒目 |
| 次要 | 副标题、重要信息 | 中等强调 |
| 辅助 | 正文、说明文字 | 适度弱化 |
| 背景 | 装饰、底图 | 最弱、不干扰 |

### 设计原则

- **对比**: 创造视觉差异，引导注意力
- **重复**: 保持一致性，建立节奏
- **对齐**: 创造秩序，提升专业感
- **亲密**: 相关元素靠近，建立关联

### 配色方法

```
60-30-10 法则:
├── 60% 主色调（背景、大面积）
├── 30% 辅助色（次要元素）
└── 10% 强调色（CTA、重点）
```

## UI/UX Guidelines

### 交互设计原则

- **可发现性**: 功能易于找到
- **反馈**: 操作有即时响应
- **容错**: 允许撤销和修正
- **一致性**: 相似功能相似操作
- **效率**: 减少用户操作步骤

### 响应式断点

| 设备 | 宽度 | 布局特点 |
|------|------|----------|
| 手机 | <768px | 单列、全宽 |
| 平板 | 768-1024px | 两列、紧凑 |
| 桌面 | >1024px | 多列、宽松 |

## Examples

```
需求: 设计一个 AI 对话产品的聊天界面

设计方案:

1. 整体风格
   - 调性: 专业、友好、现代
   - 配色: 浅色底 + 品牌蓝强调
   - 字体: 无衬线、良好可读性

2. 布局设计
   ┌─────────────────────────────┐
   │  Logo    会话标题    设置  │  Header
   ├─────────────────────────────┤
   │                             │
   │  AI消息气泡（左对齐）       │
   │                             │
   │       用户消息气泡（右对齐）│  Chat Area
   │                             │
   │  AI正在输入...              │
   │                             │
   ├─────────────────────────────┤
   │  📎  输入框         发送  │  Input
   └─────────────────────────────┘

3. 组件设计
   - 消息气泡: 圆角16px, 最大宽度70%
   - 输入框: 带placeholder, 自适应高度
   - 发送按钮: 品牌色, 禁用态灰色

4. 交互细节
   - 新消息自动滚动到底部
   - 支持 Shift+Enter 换行
   - 发送后清空输入框
```

## Design Specifications

### 间距系统

```
基础单位: 8px
├── xs: 4px   (紧凑元素)
├── sm: 8px   (相关元素)
├── md: 16px  (组内间距)
├── lg: 24px  (组间间距)
├── xl: 32px  (区块间距)
└── xxl: 48px (大区块间距)
```

### 圆角规范

- 小元素（按钮、标签）: 4-8px
- 中等元素（卡片、输入框）: 8-12px
- 大元素（模态框、容器）: 12-16px

## Safety Checks

- 确保足够的颜色对比度（WCAG 标准）
- 考虑色盲用户的可访问性
- 避免仅用颜色传达信息
- 确保可点击区域足够大（最小44px）

## Success Criteria

- 视觉设计美观、专业
- 符合品牌调性和规范
- 用户体验流畅直观
- 响应式适配良好
- 设计规范清晰完整
